# Point-Game

## 简介
基于 [Python](https://www.python.org) 使用 [Pygame](https://www.pygame.org) 库编写的二维平面游戏
**宗旨：打造一个简洁，好上手二维射击游戏**

## 依赖
[Python3](https://www.python.org) 和 [Pygame](https://www.pygame.org)

安装 Pygame
```shell
$ pip install pygame
```

## 运行
Windows
```shell
$ python main.py
```

Linux 或 MacOS
```shell
$ python3 main.py
```
